HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Header elements appear at the top of the page and will usually contain a navigational menu.
  • Paragraph elements appear in the block where the p element is nested.
  • Img element will display images when the "src" function is given a url. "alt" will be used to give a description of the image.
  • The ul element represents an unordered list via bullet point.
  • The ol element represents an ordered list via numerals.
  • The br element creates a line break between two blocks of content.
  • The anchor element creates a link to the same webpage or creates a new tab.
  • The href element is the location the url is stationed to.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • 3 Ways of CSS: Incline CSS, Internal CSS Style Sheet, External CSS Style Sheet.
  • Declarations contain the CSS property we want to apply and the value of the property.
  • Selectors used before a declaration satement represents which class element will recive the properties.
  • The text-align assignment can be used to center values.
  • The auto assignment will automatically assign left and right px of its top and bottom values.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: commits all files in the working branch
  • git pull origin main: pulls the base branch into the feature branch
  • git push: pushes local commits to the master branch
  • git config: is used to set user-specific values such as username and email

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • if statements are used for decision making in our code
  • === is used to check if two values are equal
  • An array is a single variable that is used to hold a group of data
  • Strings need to be contained in quotation marks
  • The location of data in an array is identified by a unique number or index
  • x++ is a shorter way of writing x+1
